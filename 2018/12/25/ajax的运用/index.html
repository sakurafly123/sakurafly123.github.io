1ajax的简单理解和运用

ajax全称Asynchronous JavaScript and XML
用于JavaScript执行异步网络请求，但我们在登录注册时，我们只是想验证用户名在数据库中是否存在，但是我们不想重新提交一次request，这样会刷新整个页面，我们只是想更新局部，此时我们就可以使用ajax这种方式
ajax其实就是xmlhttprequest请求

###2.3使用get还是post

  • 无法使用缓存文件（更新服务器上的文件或数据库）
  • 发送未知字符（密码)pOST 比 GET 更稳定也更可靠,GET通过 String uid = req.getParameter(“username”)。
  • 数据量比较大时，post没有限制

###2.4readyState状态值和onreadystatechange事件
|readyState|存有 XMLHttpRequest 的状态。从 0 到 4 发生变化。|
| ——– | —– |
|0|请求未初始化|
|1| 服务器连接已建立
|2| 请求已接收
|3| 请求处理中
|4| 请求已完成，且响应已就绪
|status|200:”Ok” 404:”未找到页面”|
每当 readyState 改变时，就会触发 onreadystatechange 事件。
